# ✅ Checklist de Deploy en Aternos

Seguir este checklist en orden antes del lanzamiento.

## Pre-Deploy
- [ ] Tener cuenta Aternos creada
- [ ] Servidor creado como "Bedrock" con PocketMine-MP
- [ ] Plugin ArgentinaCraft_Core compilado/listo

## Archivos a Subir
- [ ] `server/pocketmine.yml`
- [ ] `server/server.properties`
- [ ] `plugins/ArgentinaCraft_Core/` (completo)

## Configuración
- [ ] max-players configurado (≤10 para Aternos gratis)
- [ ] view-distance = 6
- [ ] difficulty = peaceful
- [ ] gamemode = adventure

## Testing
- [ ] Conectar con cuenta de prueba
- [ ] Verificar mensaje de bienvenida
- [ ] Probar /hub
- [ ] Probar /provincia buenos_aires
- [ ] Probar /pasaporte
- [ ] Verificar teleportes desde provincias al HUB
- [ ] Verificar NPCs responden
- [ ] Probar desde mobile (Android o iOS)

## Post-Deploy
- [ ] Anunciar IP del servidor
- [ ] Documentar la IP en README.md
- [ ] Invitar testers iniciales
- [ ] Monitorear TPS durante primera hora
